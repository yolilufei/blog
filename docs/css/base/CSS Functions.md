## minmax()
minmax() 函数用于定义允许的最小和最大值范围之间进行选择。minmax() 函数的语法如下：minmax(minimum, maximum)
minmax() 函数通常用于 CSS Grid 布局和弹性布局中，以设置轨道（track）的大小范围。

minmax() 只能应用到CSS Grid 布局中的尺寸指定场景。
minmax() 不能应用于 flex 布局中。
